Funcionalidades

O que o Sistema Faz

📊

Gestão de Portfólio

Acompanhe ações, renda fixa e criptoativos em BRL, USD e EUR. Cálculo de custo médio via FIFO, preços em tempo real via Yahoo Finance.

📈

Performance Analytics (GIPS)

Retornos ponderados pelo dinheiro (MWR) e pelo tempo (TWR), análise de atribuição, decomposição de performance e auditoria completa.

💸

Finanças Pessoais

Fluxo de caixa mensal, controle de assinaturas recorrentes e parcelamentos, com visualizações interativas.

🤖

Agente de IA (Gemini)

Chat inteligente com contexto do portfólio, snapshot de mercado e resumo de notícias. Usa Gemini 3 Flash Preview com fallback automático.

📰

Notícias & Mercados

Feed do Google News via RSS, dados do Yahoo Finance e integração com a API Polymarket para mercados de previsão macro, crypto, geopolítica e tech.

🔐

Autenticação & Segurança

Sistema de login com token diário. Credenciais armazenadas via Streamlit Secrets. Editor de dados com controle de acesso.

Fluxo de Dados

Como os Dados Fluem

☁️

Fontes de Dados

Google Sheets (persistência), Yahoo Finance (preços), Polymarket (previsões), Google News RSS (notícias) e PTAX (câmbio oficial).

⚙️

Processamento

Engine vetorizado em Pandas/NumPy processa transações, calcula custo médio FIFO, consolida moedas e computa métricas de performance.

🖥️

Visualização

Streamlit renderiza o dashboard com gráficos Plotly interativos, design glass-morphism e gradientes animados. Responsivo para mobile.
